## Evaluation
- We use the evaluation code from [MEFB](https://github.com/xingchenzhang/MEFB).
- We use a pytorch implementation of [MEF-SSIM](https://github.com/ChuangbinC/pytorch-MEF_SSIM) to optimize and evaluate our results.
